****🚀 AUTOHAND – AI Desktop Automation Agent**


**📌 Overview****

AUTOHAND is an AI-powered desktop automation agent that performs tasks like file creation, Excel operations, report generation, and email automation using natural language commands. It acts as a virtual assistant to improve productivity and reduce manual effort, especially in healthcare workflows.


**🎯 Features**


🤖 AI-based content & code generation

📊 Excel automation (create, update, analyze)

📄 File creation (notes, reports, scripts)

📧 Email automation

💻 Application control (VS Code, Excel, etc.)

🧠 Intelligent command understanding
